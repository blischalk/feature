    permissions() click to toggle source

    permissions should provide an array of hashes. each hash should have the keys: can, action, and resource.

    The “can” key should contain a boolean value of true or false e.g can or cannot perform some action.

    The “action” key should contain a symbol representing the action that should or should not be performed on the resource.

    The “resource” should be a string representation of the resource that this permission is being created to protect.

    These keys map to the syntax of the cancan authorization gem.

    An example return value of permissions may be:

    [{can: true, action: :read, resource: 'Post'}, {can: false, action: :manage, resource: 'Address'}]
